# buf安装

buf是protobuf的工具链与生态系统，让protobuf简单安全开发

```
scoop install buf
```

## 入门cli

### 安装示例

```
git clone git@github.com:bufbuild/buf-examples.git
cd buf-examples/cli/quickstart/start/
```

git@github.com 使用ssh协议连接github，冒号后是组织名，仓库归属人，斜杠后是仓库名

start是进行修改的地方，finish用于存放最终状态，用于对比

### 配置workspace

```
buf config init
```

然后出现yaml文件

更改为

```
version: v2
modules:
  - path: proto
lint:
  use:
    - STANDARD
breaking:
  use:
    - FILE

```

将模块指向proto子目录

```
buf build
```

### generate

先创建buf.gen.yaml文件

```
version: v2
managed://开启托管模式
  enabled: true
  override:
    - file_option: go_package_prefix
      value: github.com/bufbuild/buf-examples/gen
plugins:
  - remote: buf.build/protocolbuffers/go  //安装go插件
    out: gen  //生成的代码统一输出到项目根目录下的gen/文件夹
    opt: paths=source_relative //保持源码相对路径
  - remote: buf.build/connectrpc/gosimple  //安装connectrpc插件，生成connect风格的go服务代码
    out: gen
    opt:
      - paths=source_relative
      - simple
inputs:
  - directory: proto  //从proto目录读取
```

然后生成

```
buf generate
```

## 入门bsr

### 添加Protovalidate依赖

```
version: v2
modules:
  - path: proto
deps:
  - buf.build/bufbuild/protovalidate
lint:
  use:
    - STANDARD
breaking:
  use:
    - FILE

```

deps和go.mod差不多

要在有yaml文件的目录里update

然后在.proto文件中导入Protovalidate

```
import "buf/validate/validate.proto";
repeated LineItem line_items = 3 [(buf.validate.field).repeated.min_items = 1];
```

repeated 列表或数组，这个规则表示至少一条

### 生成代码，测试修复

```
version: v2
inputs:
  - directory: proto
plugins:
  - remote: buf.build/protocolbuffers/go:v1.36.5
    out: gen
    opt:
      - paths=source_relative
  - remote: buf.build/connectrpc/gosimple:v1.19.0
    out: gen
    opt:
      - paths=source_relative
      - simple
managed:
  enabled: true
  override:
    - file_option: go_package_prefix
      value: github.com/bufbuild/buf-examples/bsr/quickstart/server/gen
  disable:
    - file_option: go_package
      module: buf.build/bufbuild/protovalidate
```



```
buf generate
```

生成gen目录，然后启动服务器

```
go run cmd/main.go
```

### 发布共享module

创建组织

```
buf registry organization create buf.build/julien123-quickstart
Created buf.build/julien123-quickstart.
```

创建common仓库

```
buf registry module create buf.build/julien123-quickstart/common --visibility public
```

在server旁边创建同级目录

```
cd ..
mkdir common && cd common
buf config init
code buf.yaml
添加模块路径
move ..\server\proto\tag .\proto\    （注意tag后面不要加\,因为要移动v1目录，不是文件）
buf build
buf push

```
